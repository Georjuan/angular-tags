angular-tags

Pure AngularJS tagging widget with typeahead support courtesy of ui-bootstrap.

What's angular-tags?

angular-tags is a directive for AngularJS which allows you to enter "tags" into an input box. angular-tags is a pure AngularJS widget; the only prerequisites are AngularJS and UI Bootstrap.

Example

Markup for the above: 

        <tags data-options="{addable: true, classes: {white: 'white', red: 'red'}}"
              data-src="s as s.name for s in sourceTags"
              data-model="tags"></tags>

Installation

Via Bower (recommended):

bower install angular-tags

Otherwise, clone or download a tarball from GitHub
